# 💳 Premium 3D Credit Card UI

A modern and interactive **Premium 3D Credit Card UI** built using **HTML, CSS, and JavaScript**. This project simulates a professional banking card experience with **dynamic card detection**, **premium UI animations**, and a **3D hover effect**.

---

## 🚀 Features

- 💳 **Dynamic Card Detection**
  - Enter a card number starting with **5** → Mastercard automatically appears
  - Enter a card number starting with **4** → Visa card automatically appears

- ✨ **Premium 3D Hover Effect**
  - Interactive card hover animation for a modern premium feel

- 🎨 **Modern UI/UX**
  - Glassmorphism inspired premium design
  - Smooth gradients and styling

- 📱 **Responsive Design**
  - Works across different screen sizes

- ⚡ **JavaScript Functionality**
  - Real-time card type recognition

---

## 🛠️ Technologies Used

- **HTML5** – Website structure  
- **CSS3** – Styling, animations & 3D effects  
- **JavaScript** – Card logic & interactivity

## ⚙️ Installation & Usage

1. Clone the repository

```bash
git clone https://github.com/Angad198/Premium-3D-Credit-Card-UI.git
```

2. Open project folder

```bash
cd Premium-3D-Credit-Card-UI
```

3. Run project

- Open `index.html` in browser  
OR  
- Run with **Live Server** in VS Code
